InjectHookLib v0.1 by Waliedassar

waliedassar@gmail.com
http://code.google.com/p/injecthooklib/

An OllyDbg v1.10 Plugin to enable hooking system calls from User-mode. So far, Only Windows 7 (64-bit) SP0/SP1 and Windows XP SP2/SP3 are supported.
For more information about how to write hook libraries for this plugin, you can refer to http://waleedassar.blogspot.com/2012/07/wow64-user-mode-system-calls-hooking.html
