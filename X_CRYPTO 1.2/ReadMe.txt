X_CRYPTO v1.2 readme -Date 01/01/2009


X_CRYPTO allow you calculate comman hash(File & String) and Encoding.
For String(Input can be String or Hex):
MD2/4/5 - SHA1/256/384/512 - HAVAL128 -TIGER128 -
 RIPEMD 128/160 - CRC32,CRC32B - ADLER32 - GOST
Encode/Decode Base64 , Base32 , Base16, Base2 ;)
For File:
MD 2/4/5- SHA1/256/384/512 - HAVAL128/160/192/224/256 -
TIGER128/160/192- RIPEMD 128/160 - CRC32,CRC32B - ADLER32 - GOST
