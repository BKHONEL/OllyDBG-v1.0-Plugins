Olly More Menu v1.3b (Ollydbg 1.10 Plugin)

This plugin added in Ollydbg menubar more menu´s
with your favorite tools for quickstart.

Install:
   Copy this plugin in Olly/plugin Folder

-  add in OllyDbg.ini your tool path for use relative path

   [OllyMoreMenu]
   Toolpath=\Tools\


-  for add new menuentry go in add menu
   and add you favorite tools if ok add this plugin new menu´s in ollydbg
   menubar for quickstart

Bugfix:

-  If Mdi-Child active not  correctly delete the menu
